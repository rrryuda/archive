Blind onboarding 2.0

Overview

With the 3.0 update, the majority of Blind's interface underwent a transformation with new visuals and improved user experience. However, the writing screen was not included in this update. Therefore, based on research of past data, we worked on enhancing the user experience by removing unnecessary elements and adapting the new GUI to align with the 3.0 design update.

Blind 2.0

The writing experience on the existing Blind allowed users to pre-select the type of post. Upon analyzing past data, it was evident that the proportion of 'Link' type posts was remarkably low, and posts including links did not provide meaningful data. As a result, we improved the writing interface by removing the UI prompt to select the post type upon entering the writing screen. Instead, users are now immediately directed to the writing interface, streamlining the experience and eliminating the need for a separate post type selection step.
